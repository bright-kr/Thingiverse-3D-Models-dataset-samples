# Thingiverse 3D Models 데이터셋 샘플

<h2>2,923개 레코드로 구성된 샘플 데이터셋</h2>

이 **Thingiverse 3D Models Dataset Sample**에는 **2,923개 레코드**가 포함되어 있으며, **Bright Data API**를 사용하여 추출되었습니다.

## Data Points

| Field Name | Description |
|------------|-------------|
| `url` | Thingiverse의 특정 3D 모델로 바로 연결되는 직접 URL입니다. |
| `id` | 데이터셋에서 각 3D 모델 항목을 식별하는 고유 식별자입니다. |
| `domain` | 3D 모델 데이터가 수집되는 출처의 도메인 이름입니다. |
| `title` | 제작자가 3D 모델에 부여한 제목 또는 이름입니다. |
| `description` | 3D 모델에 대한 상세 설명으로, 종종 목적 또는 영감에 대한 내용이 포함됩니다. |
| `license` | 3D 모델이 배포되는 Creative Commons 라이선스 유형입니다. |
| `images` | 3D 모델 이미지로 연결되는 URL 목록입니다. |
| `creator_details` | 3D 모델 제작자에 대한 정보로, 이름 및 프로필 등이 포함될 수 있습니다. |
| `tags` | 분류를 위해 3D 모델에 연결된 키워드 또는 태그입니다. |
| `date_posted` | 플랫폼에 3D 모델이 게시된 날짜입니다. |
| `download_url` | 3D 모델 파일을 다운로드할 수 있는 URL입니다. |
| `num_comments` | 다른 사용자가 3D 모델에 작성한 댓글 수입니다. |
| `num_likes` | 3D 모델이 받은 좋아요 또는 추천 수입니다. |
| `num_files` | 3D 모델 패키지에 포함된 전체 파일 수입니다. |
| `print_settings` | 재료 및 출력 속도 등 3D 출력에 권장되는 구체적인 설정입니다. |


## Use Cases

### 1. 3D Printing Research
인기 있는 3D 모델과 해당 속성을 분석하여 3D 출력 디자인의 트렌드와 제작자 선호도를 파악합니다.

### 2. Design Education
Creative Commons 3D 모델의 데이터를 디자인 강의에서 교육 목적으로 활용하여, 실제 사례를 통해 학습을 강화합니다.

### 3. Creative Commons Licensing Analysis
3D 모델에서 사용되는 라이선스의 분포와 유형을 검토하여, 라이선스 선호도와 제작자에게 미치는 영향을 이해합니다.



## File Formats

데이터셋은 여러 형식으로 제공됩니다:
- **CSV** (이 샘플에 포함됨)
- **JSON**
- **NDJSON**
- **JSON Lines**
- **Parquet**
- **Compressed (.gz)**
